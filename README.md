# Email-_integrated_Keylogger

**WARNING**: This tool logs keystrokes and emails them.  
**Use only on devices you own or have explicit permission to monitor.**  
Unauthorized use is **illegal**.

---

## Features

- Logs every key with timestamp
- Special keys: `[SPACE]`, `[ENTER]`, `[TAB]`, `[BACKSPACE]`, `[ESC]`
- Press **ESC** → email log → delete file → stop
- Secure: **no hardcoded password**

---
